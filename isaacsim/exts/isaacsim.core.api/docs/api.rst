Objects
^^^^^^^

.. list-table::
    :header-rows: 1

    * - Type
      - Collider API
      - Rigid Body API
    * - Visual
      - No
      - No
    * - Fixed
      - Yes
      - No
    * - Dynamic
      - Yes
      - Yes
